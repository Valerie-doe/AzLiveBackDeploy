# Checklist rapide — azlivemtxn.up.railway.app en 502 + backend "Name or service not known"

## Symptômes diagnostiqués

1. `https://azlivemtxn.up.railway.app/` → **502 Application failed to respond**
   → MediaMTX n'écoute PAS sur le port public Railway (`$PORT`).
2. Backend live `webrtc.status=error` :
   `MediaMTX injoignable: Name or service not known`
   → `MEDIAMTX_API_URL` pointe vers un hostname privé **incorrect**.

Le live Facebook Graph est bien créé (stream_url présent), mais sans MediaMTX aucune vidéo n'arrive.

## A. Corriger le service MediaMTX

### Settings service MediaMTX
- Root Directory: `mediamtx/railway`
- Builder: **Dockerfile** (pas l'image Docker Hub seule)
- Networking → Generate Domain: `azlivemtxn.up.railway.app`
- **Ne force pas** un Target Port différent de `$PORT` (laisse Railway / laisse vide)

### Variables MediaMTX (obligatoires)
```env
MEDIAMTX_PUBLIC_HOST=azlivemtxn.up.railway.app
MEDIAMTX_AUTH_URL=http://azliveback.railway.internal:8080/api/media/auth/
```

Adapte `azliveback` = **nom exact** du service backend (Railway → service → Settings → nom).
Adapte `8080` = port d'écoute du backend. Sur le backend, fixe de préférence :
```env
PORT=8080
```

Redeploy MediaMTX après push du Dockerfile corrigé.

### Test OK
```bash
curl -I https://azlivemtxn.up.railway.app/
```
Tu ne dois **plus** avoir 502 (404 MediaMTX est OK).

## B. Corriger le backend Django

Dans le service backend :
```env
MEDIAMTX_ENABLED=true
MEDIAMTX_API_URL=http://NOMSERVICE.railway.internal:9997
MEDIAMTX_WHIP_BASE_URL=https://azlivemtxn.up.railway.app
MEDIAMTX_RTSP_HOST=127.0.0.1:8554
```

`NOMSERVICE` = nom **exact** du service MediaMTX dans Railway (ex. `azlivemtxn`, pas `azlivemtx`).

Pour le trouver : Railway → service MediaMTX → Settings → *Service name*,
ou onglet Networking → *Private Networking* hostname.

Redeploy backend.

## C. Retester
1. Arrête les lives `en_cours` restants dans AZLive
2. Relance un live Facebook
3. Backend `webrtc.status` doit être `"ready"` avec un `whip_url` du type :
   `https://azlivemtxn.up.railway.app/live_.../whip`
